# Exercise Tracker

Exercise Tracker microservice implemented on FastAPI.

## Instructions

To start this app:

- Installing Dependencies

```
poetry install
```

- Activating the virtual environment:

```
poetry shell
```

- Running FastApi Application:

```
uvicorn app.main:app --reload
```

- Run FastApi Test Application:

```
pytest
```
